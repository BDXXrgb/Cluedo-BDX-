    Commandes Générales :

    Objectif : Trouver quel suspect a commis le crime, avec quelle arme, et dans quel lieu.

    Déroulement d'un tour :
    • 1. Cliquez sur Lancer les Dés pour obtenir vos points de mouvements.
    • 2. Cliquez sur l'une des cases du plateau pour vous déplacer (chaque case de distance consomme 3 points).
    • 3. Faites une Hypothèse (Demande) : le lieu est automatiquement bloqué sur votre pièce actuelle. Le script interrogera les autres joueurs. Si l'un d'eux possède une de vos cartes demandées, elle vous sera secrètement révélée !
    🚨 L'Accusation Ultime : Si vous pensez avoir résolu l'énigme complète, portez une accusation.
    • Si c'est juste : Vous gagnez immédiatement la partie !
    • Si c'est faux : Vous êtes définitivement éliminé et vos dés sont bloqués, mais vos cartes restent en jeu pour contredire les autres.
